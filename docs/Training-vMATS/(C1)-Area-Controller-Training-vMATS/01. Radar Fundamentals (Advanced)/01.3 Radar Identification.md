# 01.3 Radar Identification

## 1. Objective

To positively identify an aircraft prior to providing radar control.

---

## 2. Identification Methods

### 2.1 Squawk Identification
- Assign discrete code
- Observe code change

### 2.2 IDENT Feature
- Pilot activates IDENT
- Radar return highlights

### 2.3 Position Identification
- Correlate reported position with radar return

### 2.4 Direct Recognition
- Known departure
- Transfer from another controller

---

## 3. Verification Requirements

Controllers must confirm:

- Callsign matches radar label
- Position is consistent
- No ambiguity exists

---

## 4. Loss of Identification

If identification is lost:

- Inform pilot immediately
- Re-establish identification
- Apply procedural separation if required
